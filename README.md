# Bononi Loja — Dashboard Loja Física

Dashboard operacional para pátio e tapeçaria da Bononi Acessórios.

## Stack
- React + Vite + TypeScript
- Tailwind CSS + shadcn/ui
- Supabase (banco de dados)
- Vercel (deploy)

## Rotas
- `/vendas` — Visão geral de vendas
- `/vendas/vendedores` — Ranking de vendedores
- `/servicos` — Resumo de serviços
- `/servicos/patio` — Dashboard Pátio
- `/servicos/tapecaria` — Dashboard Tapeçaria
- `/servicos/config-colaboradores` — Configuração de setores
